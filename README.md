## Portfolio

This repository contains my personal portfolio, showcasing a collection of projects built across artificial intelligence, systems programming, and modern web development. It reflects my focus on building practical, scalable, and visually refined applications.

The work presented here ranges from AI-driven tools and local model integrations to system-level utilities and full-stack web applications. Each project is designed with an emphasis on clean architecture, performance, and usability, while also exploring experimental ideas and creative implementations.

Key areas of focus include:
- Development of AI-powered tools using local and API-based models
- System-oriented projects involving automation, terminals, and utilities
- Frontend and full-stack web applications with modern frameworks
- Exploration of user experience, aesthetics, and interaction design

This portfolio is continuously updated as I build and experiment with new ideas, technologies, and workflows.****
