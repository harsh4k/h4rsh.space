## Setup

```bash
# clone the repo
git clone https://github.com/your-username/your-repo.git

# go into folder
cd your-repo

# install dependencies
npm install
# or
pip install -r requirements.txt

# run project
npm run dev
# or
python main.py
```

> Note: Setup may vary slightly depending on the project.
